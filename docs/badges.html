Badge Designer & Printing

Design your club's CR80 ID card template once — then print personalized badges for each member with one click.

Overview

RC Flight Operations includes a drag-and-drop badge designer for standard CR80 credit-card sized ID cards (85.6 × 53.98 mm). You design one template per club; every member's badge is printed from that template filled in with their data.

The designer is powered by Fabric.js (free, open-source). No special software or plugin needed — it runs entirely in your browser.

Access the designer at Administration → Badge design (Editor or Admin role required).

Badge Designer Interface

The designer has two columns:

  • Left sidebar — field panel (click any field to add it to the canvas) and a property panel that appears when an object is selected (font size, bold, italic, color, alignment).
  • Right column — the canvas area with Front and Back tabs at the top.

Click any object on the canvas to select it, then drag to reposition or use the property panel to style it. Right-click an object to delete it.

Designing the Front

  1. Click the Front tab in the canvas column.
  2. Optionally set a background image — upload a PNG/JPEG to use as the card background.
  3. Add a Static image for your club logo.
  4. Add a Photo field for the member headshot (position it where you want faces).
  5. Add text fields: member name, AMA number, renewal year — whatever your club puts on the front.
  6. Style each field using the property panel: font size, bold, color, alignment.
  7. Choose the card orientation: Landscape (horizontal) or Portrait (vertical).
Use the Live member preview dropdown to select a real member and see exactly how their card will look before saving.

Designing the Back

Click the Back tab. The badge back uses a simple HTML editor rather than the canvas designer — this makes it easy to format address blocks, add club rules text, emergency contact info, or a QR code placeholder.

You can type plain text or HTML. Common uses for the back:

  • Member's mailing address (use the {{address_block}} merge tag)
  • Club field rules or safety reminders
  • Club website and contact info
  • AMA charter number

The back orientation (landscape/portrait) is set independently from the front.

Live Member Preview

In the designer, find the Preview member dropdown at the top of the canvas area. Select any active member and the canvas updates to show their real data — name, AMA number, photo, etc. — so you can see exactly how the card will look.

Preview mode is display only. Clicking Save Template while previewing a member still saves your design layout, not the member's specific data.

Saving the Template

Click Save template. The template is stored for your club in the database. There is one template per club; saving overwrites the previous design.

There is no version history for badge templates. If you want to keep the old design, export a screenshot or copy the template JSON before saving a new version.

Printing a Badge

  1. Open the member's record from the Members list.
  2. Click Print card (visible to Editor and Admin roles).
  3. The print preview opens in a new tab showing the front and/or back of the card. Use the toggle at the top to choose Front only, Back only, or Both.
  4. Use your browser's print function (Ctrl+P / ⌘+P).
  5. Select your card printer. Set paper size to CR80 (or "Credit Card") if your printer supports it, otherwise use the card stock sheet size your printer expects.
  6. Print. The system records the badge as printed for this member.

Printing Tips

  • Fargo / Zebra card printers: These printers have their own driver software. Print from the browser to the driver; set the page size in the driver to CR80 (3.375" × 2.125").
  • Inkjet/laser on card stock sheets: Use Avery or similar CR80 blank card sheets. Match the sheet size in print settings; use "Fit to page" off so the card prints at exact size.
  • Front and back as separate jobs: Print the front, flip the cards, then print back only. The app lets you print each side independently.
  • Badge not showing member photo? Make sure the member has a photo uploaded on their Contact tab, and your template has a Photo field placed on the front canvas.
  • "Badges unprinted" dashboard card: After renewal, a member's badge is flagged as unprinted. This counter resets to zero once you print their card.
