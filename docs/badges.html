Badge Designer & Printing

Design one or more CR80 ID card templates for your club — then print personalized badges for each member with one click.

Overview

RC Flight Operations includes a drag-and-drop badge designer for standard CR80 credit-card sized ID cards (85.6 × 53.98 mm). You can save multiple named designs (for example, a standard member card and a special-event layout). Each member's badge is printed from a design you choose, filled in with their data.

The designer is powered by Fabric.js (free, open-source). No special software or plugin needed — it runs entirely in your browser.

Access the designer at Badge Design in the main navigation (Membership Manager or Administrator role required).

Multiple Designs

Use the Design dropdown at the top of the designer to switch between saved templates. Use ＋ New, Rename, and Delete to manage designs (you must keep at least one).

  • Default design — Check Default design before clicking Save Design. The default is auto-selected when printing unless you pick a different design on the print page.
  • Per-design background — Each design stores its own front background image under uploads/branding/. Uploading a new background for one design does not affect others.
  • Design picker when printing — On the print page, choose which saved design to use before printing. If you do not choose, the default design is used.

Badge Designer Interface

The designer uses a two-column layout that matches other tool pages in the app (reports, configuration):

  • Left sidebar — tabbed tool panels: Card options, Add fields, Selected field (when something is selected), and Live preview.
  • Right column — the front canvas or back HTML editor, with view zoom controls (Fit, 1×, 1.5×, 2×). Your zoom choice is remembered for the session.

Click any object on the canvas to select it, then drag to reposition. The Selected field tab opens automatically so you can style the selection.

Designing the Front

  1. In the sidebar, set Side to Front (under Card options).
  2. Optionally set a background image for this design — upload a PNG/JPEG to use as the card background.
  3. Add a Photo field for the member headshot (position it where you want faces).
  4. Add text fields: member name, AMA number, renewal year — whatever your club puts on the front.
  5. Style each field in Selected field: font family, font size, bold, italic, color, alignment, and optional fixed width.
  6. Choose the card orientation: Landscape (horizontal) or Portrait (vertical).
Use the Live member preview dropdown to select a real member and see exactly how their card will look before saving.

Designing the Back

In the sidebar, switch Side to Back. The badge back uses an HTML editor rather than the canvas designer — this makes it easy to format address blocks, add club rules text, emergency contact info, or a QR code placeholder.

Click the field buttons above the editor to insert merge tags, or type {{field_name}} by hand. A live preview below the editor shows how the back will look, with the same Fit / 1× / 1.5× / 2× zoom controls as the front.

You can type plain text or HTML. Common uses for the back:

  • Member's mailing address (use the {{address_block}} merge tag)
  • Club field rules or safety reminders
  • Club website and contact info
  • AMA charter number

The back orientation (landscape/portrait) is set independently from the front.

Live Member Preview

Open the sidebar Live preview tab and choose a member from the dropdown. The front canvas updates to show their real data — name, AMA number, photo, etc. — so you can see exactly how the card will look.

Preview mode is display only. Clicking Save while previewing a member still saves your design layout, not the member's specific data.

Saving a Design

Set the design name with Rename (or start from ＋ New), optionally check Default design, then click Save Design in the page header. The design is stored in the database with its own background image, front canvas JSON, and back HTML.

To start a fresh layout, click ＋ New, build the card, rename it, and save. Switch between saved designs with the Design dropdown.

There is no version history for badge templates. If you want to keep the old design, save it under a different name or export a screenshot before overwriting.

Printing a Badge

  1. Open the member's record from the Members list, or finish a renewal in the renewal workflow.
  2. Click Print card (visible to Membership Manager, Administrator, and Club Staff roles).
  3. On the print page, choose a design if you have more than one saved template. The default design is pre-selected.
  4. The print preview shows the front and/or back of the card. Use the toggle at the top to choose Front only, Back only, or Both.
  5. Use your browser's print function (Ctrl+P / ⌘+P).
  6. Select your card printer. Set paper size to CR80 (or "Credit Card") if your printer supports it, otherwise use the card stock sheet size your printer expects.
  7. Print. The system records the badge as printed for this member.

Printing Tips

  • Fargo / Zebra card printers: These printers have their own driver software. Print from the browser to the driver; set the page size in the driver to CR80 (3.375" × 2.125").
  • Inkjet/laser on card stock sheets: Use Avery or similar CR80 blank card sheets. Match the sheet size in print settings; use "Fit to page" off so the card prints at exact size.
  • Front and back as separate jobs: Print the front, flip the cards, then print back only. The app lets you print each side independently.
  • Badge not showing member photo? Make sure the member has a photo on their Contact tab (JPEG, PNG, or GIF under uploads/member_photos/), and your template has a Photo field placed on the front canvas. For website applications, the photo is copied automatically when staff approve — if it is missing, upload manually or check that the applicant submitted a supported image format.
  • "Badges unprinted" dashboard card: After renewal, a member's badge is flagged as unprinted. This counter resets to zero once you print their card.
